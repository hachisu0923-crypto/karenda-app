📲 ホーム画面に追加が必要です

iPhone / iPad では、ホーム画面に追加した状態でのみ通知を受け取れます。

  1. Safari 下部の 共有ボタン（□↑） をタップ
  2. 「ホーム画面に追加」 を選択
  3. ホーム画面のアイコンからアプリを開く
  4. もう一度「🔔 通知を有効にする」をタップ
💳 JCBメール自動取込
利用通知メールの金額を家計簿へ自動登録

Gmail に届く JCB のご利用通知メールから金額を自動で家計簿に取り込めます。 設定には下の 2 つの値を使います（詳しい手順は SETUP_JCB_INGEST.md）。

手順の概要：① SQL を実行 → ② Edge Function ingest-jcb をデプロイ → ③ Secrets（INGEST_SECRET / 上記 INGEST_USER_ID）を登録 → ④ Google Apps Script を設定して 15 分ごとに実行。
